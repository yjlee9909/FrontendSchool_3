## 2. branch 만들기 및 merge test

```shell
$ git branch // 브랜치 목록 확인
$ git branch a // 브랜치 a 생성

(a) $ echo '# hello world a' >> 'hello1.txt'
(a) $ git status
(a) $ git add .
(a) $ git commit -m "a1"
(a) $ git push // error
(a) $ git push --set-upstream origin a
(a) $ git checkout main
(main) $ git branch b // 브랜치 a 생성
(main) $ git branch
```
